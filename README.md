
#  Docker Bootcamp — 5-Day Live Hands-on Course for DevOps

Welcome to the **Docker Bootcamp** by [LearnWithPrayag](https://learnwithprayag.com) — a practical, community-driven course designed for DevOps engineers, cloud professionals, and anyone eager to master Docker and Docker Compose through real-world examples.

##  What You Will Learn

| Day | Topics |
|-----|--------|
| **Day 1** | Docker Introduction, CLI, Images vs Containers |
| **Day 2** | Dockerfile, Building Images, CMD vs ENTRYPOINT |
| **Day 3** | Docker Compose, Multi-Container Apps |
| **Day 4** | Volumes, Networks, Persisting Databases |
| **Day 5** | CI/CD with Docker: GitHub Actions + Jenkins |

---

##  Hands-on Labs

- Run and inspect containers (`nginx`, `redis`, `alpine`)
- Build Dockerfiles for Node.js, Python, and Java apps
- Create multi-container stacks with Docker Compose
- Persist databases using volumes and bind mounts
- CI/CD pipeline using GitHub Actions and DockerHub
